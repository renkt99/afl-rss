AFL.com.au RSS Feed

Feed Information

This is an unofficial RSS feed for AFL.com.au news articles. The feed is automatically updated every 6 hours and excludes AFLW content by default.

Feed URL

https://[your-username].github.io/afl-rss/feed.xml

Features

  • Updates every 6 hours
  • Contains only articles from the past 24 hours
  • Excludes AFLW articles by default
  • Standard RSS 2.0 format

Usage

To use this feed in your RSS reader, simply add the feed URL above to your reader of choice.

View Raw Feed View on GitHub

Configuration

The feed can be configured by modifying the config.yml file in the repository. You can customize:

  • URL patterns to exclude
  • Title keywords to exclude

This is an unofficial feed and is not affiliated with AFL.com.au.
